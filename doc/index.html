Install Template

After unzip the download pack, you'll found a Template Folder with all the files.

To use this project, you'll need the following things installed on your machine.

1. [Jekyll] (http://jekyllrb.com/) - $ gem install jekyll bundler

Start Jekyll server on your local machine $ Bundle exec jekyll serve Injoy

Basic

To keep your sanity and better manage dependencies I strongly urge you to install Bundler with gem install bundler and use the following Gemfile:

You can view this Template in any browser, you can display or edit the Template without an internet connection.(May not wotrk fonts and google map).

Colors Schemes

Just change on you ./_config.yml file. "(green.css blue.css, orange.css, red.css, pink.css, purple.css)" 

					
color_scheme: "green" # "green" (default), "blue", "orange", "pink", "purple", "red"

./includes/head.html

					
<!-- CSS -->
<link rel="stylesheet" href="{{ '/css/basic.css'| prepend: site.color_scheme | prepend: 'assets/'  | relative_url }}" />
<link rel="stylesheet" href="{{ '/css/layout.css'| prepend: site.color_scheme| prepend: 'assets/'   | relative_url }}" />
<link rel="stylesheet" href="{{ '/css/ionicons.css'| prepend: site.color_scheme| prepend: 'assets/'   | relative_url }}" />
<link rel="stylesheet" href="{{ '/css/magnific-popup.css'| prepend: site.color_scheme| prepend: 'assets/'   | relative_url }}" />
<link rel="stylesheet" href="{{ '/css/animate.css'| prepend: site.color_scheme| prepend: 'assets/'   | relative_url }}" />

Background Style or Home Card

4 Background Styles (Image, Color, Particles, Video)

Your need to change ./_config.yml file

					
home_style:  'background-color' # "background-color" (default), "particles-bg", "video-bg"

./includes/sections/start.html

					
<!-- Section Started -->
<div class="section started">
{%- if site.home_style == 'background-color' -%}
	<div class="slide" style="background-image: url({{ 'images/slide-bg-1.jpg' | relative_url }});"></div>
{%- elsif site.home_style == 'particles-bg' -%}
<div id="particles-bg" class="slide" style="background-image: url({{ 'images/slide-bg-1.jpg' | relative_url }});"></div>
{%- elsif site.home_style == 'video-bg' -%}
<div id="video-bg" class="slide" data-property="{videoURL:'https://youtu.be/ClIBmBT9clU', containment:'#video-bg', showControls:false, autoPlay:true, loop:true, mute:true, startAt:0, opacity:1, addRaster:true, quality:'default'}"></div>
{%- endif -%}
<div class="st-box align-center">
	<div class="st-title">Hi, I'm {{ site.first_name}} {{ site.last_name}} </div>
	<div class="st-subtitle"> {{ site.profession}}</div>
	<div class="st-soc">
		{%- for social in site.social_link -%}
		<a target="blank" href="{{ social.link }}"><i class="{{ social.icon }}"></i></a>
		{%- endfor -%}
	</div>
</div>
</div>
